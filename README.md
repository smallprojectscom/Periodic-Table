# ⚛️ Periodic Table Pro Ultimate

An advanced interactive periodic table web application designed to make chemistry learning simple, visual, and engaging.

---

## 🌐 Live Demo
👉 https://smallprojectscom.github.io/periodic-table/

---

## 🚀 Features

- 🔬 Full interactive periodic table (118 elements)
- 📊 Compare two elements side-by-side
- 🧠 Flashcard learning mode
- 📚 Group-wise study mode
- 📈 Trend visualization (atomic radius, electronegativity)
- ⚛️ Animated atom model (electron shells)
- 🧾 Printable revision sheet
- 🎨 Modern responsive UI (mobile + desktop)

## 🎯 Purpose

This project is built for students (Class 11/12, JEE, NEET) to:
- Understand periodic trends visually
- Compare elements easily
- Memorize using flashcards
- Revise efficiently
